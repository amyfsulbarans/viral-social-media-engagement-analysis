# Viral Content & Social Media Engagement Analysis

## Project Overview
This project analyzes 5,000 viral social media posts across TikTok, Instagram, Twitter, and YouTube to identify engagement patterns and content performance.

The objective is to determine which platforms and content types generate the highest engagement and provide actionable insights for marketing strategies.

## Tools Used
- Python
- Pandas
- Matplotlib
- SQL (for analytical queries)

## Key Metrics
- Engagement Total (likes + shares + comments)
- Engagement Rate

## Key Insights

- Instagram shows the highest engagement rate, indicating stronger audience interaction.
- TikTok also performs strongly, confirming the effectiveness of short-form content.
- Shorts significantly outperform other content types, highlighting the importance of concise and engaging formats.

## Business Recommendations

- Focus on Instagram and TikTok for content distribution  
- Prioritize short-form video formats (Shorts)  
- Optimize content for quick consumption and high interaction  
- Test live content strategies for additional engagement
